# 📘 Database vs DBMS (Complete Notes)

---

# 🧠 1. What is a Database?

A **database** is an organized collection of data stored in a structured format so it can be easily accessed, managed, and updated.

### Example:

| ID | Name  | Age |
| -- | ----- | --- |
| 1  | Rahul | 20  |
| 2  | Neha  | 21  |

👉 A database only stores data, it does NOT manage it.

---

# 🧠 2. What is DBMS?

A **Database Management System (DBMS)** is software that is used to:

* Create databases
* Store data
* Retrieve data
* Update data
* Delete data

👉 DBMS acts as an interface between user and database.

---

# ⚖️ 3. Database vs DBMS (Main Difference)

| Feature          | Database                  | DBMS                        |
| ---------------- | ------------------------- | --------------------------- |
| Definition       | Collection of data        | Software to manage data     |
| Purpose          | Stores data               | Manages data                |
| Nature           | Passive                   | Active system               |
| User interaction | No direct control         | Provides interface          |
| Operations       | Cannot perform operations | Supports CRUD operations    |
| Security         | No built-in security      | Provides security features  |
| Backup           | Not available             | Backup & recovery supported |

---

# 🏗️ 4. Real-Life Analogy

## Database

## DBMS
