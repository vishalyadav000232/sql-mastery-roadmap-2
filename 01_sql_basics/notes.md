# 📘 SQL Basics Notes

---

# What is SQL?

SQL (Structured Query Language) is used to interact with databases.

---

# Core Operations (CRUD)

| Operation | SQL Command |
|----------|------------|
| Create | INSERT |
| Read | SELECT |
| Update | UPDATE |
| Delete | DELETE |

---

# Database Workflow

Application → SQL Query → Database → Result

---

# Key Concepts

- Database stores data
- Tables store structured data
- Rows = records
- Columns = fields

---

# Important Rules

- Always use WHERE in UPDATE/DELETE
- Use PRIMARY KEY for uniqueness
- Use meaningful column names

---

# Real-world Use

- Banking systems
- E-commerce apps
- Library systems
- Authentication systems
